# Job Fraud Detection Analysis

## 📸 Project Preview
![Project Screenshot](project_screenshot.png)

## 📌 Project Overview
This project analyzes job postings to identify patterns that distinguish fake and real job listings. The goal is to help users recognize fraudulent job opportunities and avoid scams.

## 🛠 Tools Used
- Microsoft Excel
- Data Analysis
- Data Visualization

## 🔍 Key Insights
- Fake job postings often use vague descriptions and unrealistic promises
- Many fake jobs require low qualifications and lack company details
- Real job postings include structured responsibilities and specific skill requirements

## ⚠️ Key Red Flags Identified
- No company information
- Unrealistic salary offers
- No specific skills required

## 📊 Output
- Comparison table (Fake vs Real jobs)
- Pie chart visualization
- Insights and conclusion

## 📁 Files Included
- Excel analysis file
- PDF report
- Project_screenshot.png

## 🎯 Conclusion
This project highlights how simple data analysis can help identify fraudulent job postings and improve awareness among job seekers.
